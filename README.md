## v2.0.0 "SPIRE" — Major Update

This is a complete rewrite with AAA-level polish:

- **Deterministic Replays** — Every run is fully reproducible using a seeded RNG and recorded input stream.
- **Verifiable Proofs** — Generate cryptographically signed proof PNGs with embedded run data. Old proofs remain compatible.
- **Proof Archive** — Last 12 proofs are saved locally with one-click "Submit to CNI" JSON export (ready for on-chain integration).
- **Improved Mobile Controls** — Native-feel touch zones, analog drag steering, timed laser, and swipe dash.
- **Full Gamepad Support** — Hot-plug controller support with proper button mapping.
- **24 Achievements** — Height, combo, skill-based, and lifetime goals with meaningful 🧶 and 🐟 rewards.
- **New Content** — Screen Purge consumable + Combo Keeper forge upgrade.
- **Juice & Polish** — Hitstop, screenshake, hazard telegraphs, reactive layered synthwave music, and more.
- **PWA Ready** — Installable with offline support via service worker.
